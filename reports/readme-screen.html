Funcionalidades

Tudo que voce precisa para acelerar o desenvolvimento

Um ecossistema completo de agentes que trabalham juntos para transformar ideias em software de producao.

Figma to Code

Converte automaticamente designs do Figma em codigo HTML/CSS ou React+TypeScript fiel ao pixel, com boas praticas de acessibilidade.

Audit Automatico

Audita o codigo gerado comparando com o design original, identificando diferencas visuais e pontos de melhoria.

Deploy Check

Executa checklist completo pre-deploy: build, lint, testes, tipos, dependencias e variaveis de ambiente.

DB Migration

Gerencia migrations de banco de dados com suporte a TypeORM, Prisma, Knex e Sequelize, incluindo rollback automatico.

Spec Writer

Gera especificacoes tecnicas detalhadas a partir do design do Figma, documentando componentes, fluxos e requisitos.

Git Automation

Automatiza commits semanticos, push e abertura de Pull Requests com descricoes detalhadas das alteracoes.

Pipeline

Fluxo automatizado do squad

Os agentes executam em sequencia, cada um entregando para o proximo, formando um pipeline continuo de desenvolvimento.

Agentes

Skills disponiveis no squad

Cada agente e especializado em uma tarefa e pode ser acionado individualmente ou como parte do pipeline automatizado.

figma-extract

Extrai tokens de design do Figma via MCP: cores, tipografia, espacamentos e componentes.

git-commit

Gera e executa commits semanticos baseados no diff das alteracoes pendentes.

git-push-pr

Faz push da branch e abre Pull Request automaticamente com titulo e descricao gerados.

deploy-check

Executa todos os checks necessarios antes do deploy e gera relatorio de status.

db-migration

Cria, executa e reverte migrations de banco de dados com suporte a multiplos ORMs.

code-review

Analisa o codigo em busca de bugs, vulnerabilidades e oportunidades de refatoracao.

create-issue

Cria issues no GitHub com titulo, descricao, labels e assignees adequados.

test-runner

Executa suites de testes, coleta cobertura e reporta resultados de forma estruturada.

Instalacao

Como comecar em minutos

Configure o ambiente local e execute o squad de agentes com apenas alguns comandos.

1
Clone o repositorio
Faca o clone do repositorio para sua maquina local.
git clone https://github.com/org/fusioncodeapp.git
cd fusioncodeapp
2
Instale as dependencias
Instale todos os pacotes necessarios via npm ou yarn.
npm install
3
Configure as variaveis de ambiente
Copie o arquivo de exemplo e preencha com suas credenciais.
cp .env.example .env
# edite o .env com suas chaves de API
4
Execute o squad
Inicie o pipeline completo de agentes passando a URL do Figma.
node cli/bin/fusion.js squad --cwd .
terminal 
# Clonar e configurar
$ git clone https://github.com/org/fusioncodeapp.git
$ cd fusioncodeapp
$ npm install
 
# Variaveis de ambiente
$ cp .env.example .env
$ nano .env
 
# Executar o squad
$ node cli/bin/fusion.js squad --cwd .
 
# Ou executar agente individual
$ node cli/bin/fusion.js run figma-to-code
  --url "https://figma.com/file/..."
