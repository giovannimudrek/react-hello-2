# Hello World

## Descricao

Pagina web simples de Hello World desenvolvida em HTML5, com conteudo em portugues brasileiro (pt-BR). O projeto exibe uma mensagem de boas-vindas no navegador e serve como ponto de partida para projetos web basicos.

## Tecnologias utilizadas

- HTML5
- Idioma: Portugues brasileiro (pt-BR)

## Como executar

1. Abra o arquivo `index.html` diretamente no seu navegador (Chrome, Firefox, Edge, Safari, etc.)
2. Nenhuma instalacao ou dependencia adicional e necessaria

## Estrutura do projeto

```
/
├── index.html   # Pagina principal Hello World
└── README.md    # Documentacao do projeto
```
