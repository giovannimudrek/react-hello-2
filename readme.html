Funcionalidades

Tudo que seu time precisa

Da ideacao ao deploy, o FusionCode Squad automatiza as tarefas repetitivas para que seu time foque no que importa.

🎨

Figma to Code

Converte designs do Figma diretamente em HTML e CSS pixel-perfect via MCP. Extrai cores, tipografia, espacamentos e gera codigo semantico e responsivo automaticamente.

🔍

Code Review Automatico

Analisa seguranca, qualidade, performance e convencoes do projeto. Gera relatorio com severidade, sugestoes de fix e score geral para cada revisao.

⚙

Squad Pipeline

Orquestra multiplos agentes em sequencia. Cada agente passa contexto para o proximo, formando um pipeline completo do design ao codigo auditado.

📋

Jira e GitHub

Cria e fecha issues automaticamente, sincroniza com Jira, abre Pull Requests com titulo e descricao gerados a partir dos commits da branch.

📈

Deploy Check

Executa checklist completo pre-deploy: build, lint, testes, type-check e auditoria de dependencias. Resultado: READY ou NOT READY com relatorio detalhado.

📊

DB Migrations

Detecta o ORM (TypeORM, Prisma, Knex, Sequelize) e gera migrations com UP e DOWN. Segue boas praticas de idempotencia e rollback seguro.

Setup

Como comecar

Configure o FusionCode Squad em seu projeto em poucos passos.

1

Clone o repositorio

Faca o clone do repositorio e entre no diretorio do projeto.

bash git clone https://github.com/brq/fusioncodes-squad.git
cd fusioncodes-squad
2

Configure as variaveis de ambiente

Crie o arquivo .env com suas chaves de API do Figma e Jira.

.env # Figma Personal Access Token
FIGMA_PERSONAL_ACCESS_TOKEN=figd_...

# Jira / Atlassian
JIRA_URL=https://seu-workspace.atlassian.net/
JIRA_USERNAME=seu@email.com
JIRA_API_TOKEN=ATATT3x...
3

Instale as dependencias do MCP

Instale os servidores MCP para Figma e Jira.

bash # MCP Figma
npx -y mcp-figma

# MCP Atlassian (Jira)
uvx mcp-atlassian
4

Execute o squad em uma issue

Passe o numero de uma issue do GitHub para o squad processar automaticamente.

bash # Executa o pipeline completo para a issue #42
node cli/bin/fusion.js squad 42 --cwd .

# Ou execute uma skill isolada
node cli/bin/fusion.js skill figma-extract https://figma.com/design/...
5

Acompanhe o pipeline

Os agentes geram artefatos no diretorio .pipeline/ que ficam disponiveis para os proximos agentes da sequencia.

output # Artefatos gerados automaticamente
.pipeline/
  figma-nodes.json   # dados do design
  figma-preview.md  # URL da imagem renderizada
  codebase-analysis.md
  code-review.md
  deploy-check.md
